# Natural Language to Windows CLI Agent 🤖

סוכן בינה מלאכותית המתרגם פקודות בשפה חופשית (עברית/אנגלית) לפקודות CMD של Windows בצורה בטוחה ומדויקת.

## 📋 תיאור הפרויקט
הפרויקט עושה שימוש במודל Gemini כדי לפרש כוונות משתמש ולהמיר אותן לפקודות מערכת. 
הסוכן כולל מנגנוני הגנה המזהים פקודות הרסניות ומזהירים את המשתמש לפני הביצוע.
